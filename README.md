# Clawr Documentation

[MIT License](./LICENSE) [How to contribute](./CONTRIBUTING)

Clawr is a language with goals of clarity, a modelling focus and easy refactoring. The name is a portmanteau of the word ”clarity,” and a lion’s roar.

This repository contains documentation for the Clawr language. The documentation is separated by audience:

- [Language developers](developers/README.md)
- [Language users](users/README.md)
- [Backend developers](backend/README.md)
- [Reference documentation](reference/README.md)

## Other Repositories

The main repository for Clawr development is <https://github.com/clawrlang/clawr>. There is also a Visual Studio Code extension at <https://github.com/clawrlang/vscode-extension>.
